수동 리포트 업로드
봇 자동 업로드 API

자동매매 봇에서 아래 코드를 사용하면 리포트가 자동으로 올라옵니다:

import requests

url = "https://www.ppwangga.com/api/upload"
headers = {"X-API-Key": "YOUR_API_KEY"}

# 파일 업로드
files = {"file": open("report.html", "rb")}
data = {"date": "2026-03-02"}
requests.post(url, headers=headers, files=files, data=data)

# 또는 HTML 텍스트 직접 전송
requests.post(url, headers=headers, json={
    "date": "2026-03-02",
    "html": "<html>리포트 내용</html>"
})
