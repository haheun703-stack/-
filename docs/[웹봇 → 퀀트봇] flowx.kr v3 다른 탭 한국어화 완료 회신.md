## 2. 적용 결과 상세

### 2-A. 공통 사전 — `shared/lib/quantTermsKr.ts` 신규
```typescript
COMMON_KR: { RSI:'상대 강도 지수', MA20:'20일 이동평균', BB:'볼린저밴드',
             PER:'주가수익비율', ROE:'자기자본이익률', EPS:'주당순이익',
             WR:'승률', PF:'손익비', MDD:'최대 손실폭',
             GAP:'괴리율', FIRE:'섹터 발화', ... }
abbrKr(code)        // "상대 강도 지수 (RSI)"
abbrKrShort(code)   // "상대 강도 지수"
ABBR_TOOLTIP        // 호버 툴팁용 짧은 설명
fmtKrAmount(mw)     // 백만원 → "1.5조" / "840억" / "4,500백만원"
FIRE_LEVEL_KR       // STRONG/MODERATE/WEAK/EARLY/MATURE → 한국어
FIB_LEVEL_KR        // fib_618 → "황금 되돌림 (61.8%)"
GAP_LABEL_KR        // POSITIVE_GAP/NEGATIVE_GAP → 한국어
```

**가드 충돌 회피**: `BUY: '매수'` / `SELL: '매도'` 매핑은 의도적 미포함 — 5/14 등급 리터럴 가드 차단 단어. 컴포넌트별 라벨 직접 처리.

### 2-B. SectorFireView — 발화 + 진입/관망
```typescript
BUY_GRADE_CONFIG: {
  STRONG: { label: '강력' },
  BUY:    { label: '진입' },   // 지시서 "BUY" → 우리 "진입" (5/14 표현 변경 + 가드 회피)
  WATCH:  { label: '관망' },   // 지시서 "WATCH" → 우리 "관망"
}

필터 버튼: "BUY n" → "진입 n", "WATCH n" → "관망 n"
배지: "FIRE n등급 n점" → "발화 n등급 n점"
헤더: "SECTOR FIRE" → "섹터 발화 (FIRE)" + title 툴팁
```

### 2-C. 영문 약어 헤더 한국어 병기

```html
</th>
<th title="PER: 주가수익비율 (주가 ÷ 주당순이익). 낮을수록 저평가">

<th label="ROE (자본효율)">
  ROE (자본효율)
</th>
```

→ 약어 폭 유지 (테이블 컬럼 좁음) + 학습 효과 + 호버 시 자세한 설명.

## 7. 협력 요청 (퀀트봇 측)

### 7-A. 다른 컴포넌트에 영문 약어 노출 발견 시
v3 지시서 line 252-258 약어 (RSI/MA20/BB/PER 등)가 향후 새 컴포넌트/기능 추가 시 노출되면 `shared/lib/quantTermsKr.ts` COMMON_KR에 자동 활용 가능.

### 7-B. fmtKrAmount 활용 시점
백만원 → 한국 컨벤션 (조/억/만원) 변환 헬퍼 작성 완료. 향후 sector_investor_flow API의 외인/기관 순매수 표시에 활용 가능 (현재 EtfInvestorTab의 fmtNet은 원 단위 가정).

### 7-C. FIB_LEVEL_KR / GAP_LABEL_KR 적용 여부
사전 작성됐으나 현재 사용자 화면에 fib_* / POSITIVE_GAP 등 영문 직접 노출 발견 0건 → 적용 보류. 향후 노출 발견 시 동일 사전 즉시 활용.
