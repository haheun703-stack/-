## 1. 배경 & 목적
- 사장님 관심(투자 영상 '손주부' 프레임): 시총 TOP30 우량주를 ROE/PER/PBR로 보고 "쌀 때 산다".
- 영상의 **맹점 3종을 우리가 보정**:
  1. 이익 방향 = 추정(컨센서스)PER vs 현재PER (반도체 'PER 거꾸로 함정' 자동 탐지)
  2. **FCF(잉여현금흐름)** — ROE가 진짜인지 부채로 부풀렸는지의 결정타. ROE는 분식 가능, FCF는 못 속임.
  3. 부채(순부채·D/E)
- 핵심 = "ROE 높은데 떨어진" 종목을 **옥(저점후보)과 석(가치함정)으로 자동 분리**.
  - 실증: 오라클 ROE 53%인데 **FCF −3.8%(현금 태움)** = 가치함정 / 메타 ROE 33%·**FCF +1.8%** = 저점후보.

## 2. ★현황 발견 — scan_bluechip은 고아(미가동) → 신규 권고
- `scan_bluechip_checkup.py`는 코드상 대형주 TOP30을 5축(밸류/실적모멘텀/수주/수급/기술)으로
  `quant_bluechip_checkup`에 적재하도록 돼 있으나, **실제로는 미가동**:
  - BAT 미배선(어떤 .bat에도 없음) + `data/bluechip_checkup.json` 마지막 생성 **2026-04-12**(2달 정지).
  - derivatives_collector·four_signal과 동일한 "코드는 있는데 BAT 미배선 고아" 패턴.
- ⚠️ **정보봇 주의**: `quant_bluechip_checkup` 테이블이 **4/12 이후 stale**일 가능성 높음(웹 표시 중이면 점검 요망).
- **결론: 통합 전제 깨짐 → 신규 `dashboard_valuation_band` 권고.** scan_bluechip 부활(BAT 배선 +
  freeze 검토)은 별도 트랙이며, 부활 시 5축 + FCF/옥석 통합을 재검토.

## 4. UI 스펙 — "한눈에" (목업 위젯 별첨)
**한눈에 = 3요소:**
1. **52주 밴드 막대** — 마커가 왼쪽(초록 0~35% 구간)이면 싸짐(하단), 오른쪽이면 비쌈(상단).
2. **옥석 색 배지** — 저점후보=초록 / 가치함정=빨강 / 이미오름=노랑 / 관망=회색.
3. **이익 화살표** — 추정PER 기준 ↑(이익 성장)/↓(감소).

**구성:**
- 상단: 주목 카드 3 (저점후보·가치함정 핵심).
- 메인: 종목 행 리스트 — `시장칩 | ROE | PER→추정(화살표) | FCF수익% | 52주 밴드막대 | 판정배지`.
  정렬 = 저점후보 → 가치함정 → 이미오름 → 관망.
- ★**이미오름 그룹은 추정PER로 2분할**: "천장+이익 폭증(삼성·SK·머크 — 추정PER 7배대)" vs
  "천장+평탄(코카콜라 등)". 전자는 별색 강조 = 영상의 "삼성은 천장이어도 싸다" 자리.
- 범례 필수.
