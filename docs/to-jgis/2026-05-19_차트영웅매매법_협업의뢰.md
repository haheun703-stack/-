```sql
CREATE TABLE quant_company_card (
  ticker          TEXT PRIMARY KEY,
  name            TEXT,
  sector          TEXT,
  biz_summary     TEXT,        -- 한 줄 (50자), Perplexity
  main_products   JSONB,       -- [{product, revenue_pct}], DART
  q_revenue_yoy   FLOAT,       -- 분기 매출 YoY, DART
  q_op_yoy        FLOAT,       -- 분기 영업이익 YoY, DART
  per             FLOAT,
  pbr             FLOAT,
  roe             FLOAT,
  debt_ratio      FLOAT,
  forward_per_12m FLOAT,       -- 12개월 선행 PER, FnGuide/KIS
  market_cap_억   BIGINT,
  avg_tv_5d       BIGINT,      -- 5일 평균 거래대금
  updated_at      TIMESTAMP
);
```
**갱신**: 매일 18:30 (BAT-D 직후, 수치만), biz_summary는 분기 1회

### 요청 2: 상한가 재료 분석 — `quant_surge_catalyst` (Supabase) ⭐⭐⭐
**마감**: 5/21 18:00 (퀀트봇 핵심)

```sql
CREATE TABLE quant_surge_catalyst (
  date              DATE,
  ticker            TEXT,
  name              TEXT,
  sector            TEXT,
  surge_price       INT,
  surge_pct         FLOAT,
  catalyst_summary  TEXT,           -- 재료 1줄, Perplexity
  catalyst_category TEXT,           -- AI반도체/방산/바이오/2차전지/조선/원전/양자/로봇/...
  continuity_score  FLOAT,          -- 0~100
  sector_co_surge_n INT,            -- 동일 섹터 +3% 이상 동반 종목 수
  news_count_5d     INT,            -- 5일 뉴스+공시 빈도
  smart_money_5d    FLOAT,          -- (외인+기관 5일 누적) / 시총
  is_one_off_event  BOOL,           -- 호르무즈/이란 같은 일회성?
  PRIMARY KEY (date, ticker)
);
```
**갱신**: 매일 16:30 (상한가 확정 후)
