## 0. 배경
2층 구조("나만의 ETF") 확정 — 설계서 `docs/02-design/two_layer_portfolio_design_2026-06-15.md`.
```
코어 82%(밸류밴드 우량주) + 새틀라이트 18%(미국 레버리지 B상시) + 분기 리밸런싱 + 드로다운 −15% 알림
```
영상 2개(손주부=밸류 / Direxion=레버리지) 종합 + 백테스트 검증 결과.

## A. 코어 — 밸류밴드 패널 완성 (모듈 `src/use_cases/valuation_band.py`)
- [ ] **A1. 수급 축 추가** ★최우선 — 외인/기관 순매수(`kospi_investor_flow` 보유). "싸도 수급 빠지면
  falling knife"(6/8 교훈). 영상엔 없는 우리 강점. verdict에 수급 신호 결합.
- [ ] **A2. Supabase 적재** — `dashboard_valuation_band` 테이블, UPSERT `(date,market,ticker)`,
  finality `snapshot_time`(KST·6/11 교훈 = 장중 미확정값 오염 방지), `--write` 가드(기본 dry).
- [ ] **A3. 이미오름 추정PER 2분할** — "천장+이익폭증(삼성·SK 추정PER 7배)" vs "천장+평탄(코카콜라)".
- [ ] **A4. ASML PBR 이상치 등 정제 필터.**
- 데이터 소스(재사용): 미국 yfinance(★SSL 한글경로→cacert ASCII 복사), 한국 네이버 모바일 API
  (PER/추정PER/PBR) + yfinance `.KS`(ROE/FCF/순부채/52주/시총).

## B. 새틀라이트 — 레버리지 슬리브 (신규)
- [ ] **B1. 미국 레버리지 B 상시 18%** — SOXL/TQQQ/NVDL. 유동성 필터(일거래대금 ≥50$M →
  USD/FNGU/BULZ 1~2$M 탈락). 구성 비중(6/6/6 기본, 과최적화 주의).
- [ ] **B2. 한국 레버리지는 별첨만** — 기본예탁금 1,000만원 장벽으로 미국 다이렉트 우선.

## C. 드로다운 −15% 알림 ★핵심 (신규)
- [ ] **C1. 드로다운 모니터** — `account_peak`(6/14 보유) 재활용. 계좌 고점 대비 DD −15% 트리거.
- [ ] **C2. "리먼급 판별 대시보드" 생성** (★알림 ≠ 행동 = 판단재료+권장행동 필수):
  - `stress_test`(6/14): 지금 손실이 역사 어느 폭락(2008/2020/2022)과 닮았나
  - 매크로: 신용스프레드·VIX·환율 = 시스템 위기 여부
  - 수급: 외인 이탈 규모
  - `factor_exposure`(6/15): 포트 노출(반도체 N배 등)
  - **권장 행동**: 버티기(V반등)/레버 절반/헤지/추가매수 — 각 근거 동반
- [ ] **C3. telegram 발송** (telegram 보유).
