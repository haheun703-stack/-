## 1. 본질 — 수익 극대화 4축

수익 = (사이즈) × (정확도) × (회전) × (시장 환경 적합도)

이 식의 4축을 동시에 확장하는 것이 수익 극대화 전략이다. 5/19 현재 4축 모두 보수적 초기값. 6/15+ 점진 확대.

### 1-1. 4축 정의 및 현재/목표 매트릭스

| 축 | 정의 | 측정 단위 | 5/20 (P1) | 6/15 (P3) | 7/14 (P5) | 1년 (안정) |
|---|---|---|---|---|---|---|
| **사이즈** | 단일 매수 자본 | 원 | 10만원 | 100만원 | 1,000만원 | 5,000만원+ |
| **정확도** | D+1 적중률 | % | 미측정 | 50%+ | 60%+ | 65%+ |
| **회전** | 일일 매매 건수 | 건 | 1 | 3 | 5~10 | 10~20 |
| **시장 환경 적합도** | regime별 후보 풀 다양성 | 풀 수 | 1 (강력포착만) | 3 (강세/약세/횡보) | 5 (+갭/테마) | 7 (+섹터/이벤트) |

### 1-2. 각 축의 수익 기여도 (산술)

- **사이즈 100배** (10만 → 1,000만): 수익 100배
- **정확도 1.5배** (40% → 60%): PF가 약 2.0배 (60×R-40×L = 0.5R-0.4L vs 0.4R-0.6L)
- **회전 5배** (1건 → 5건): 일 기대수익 5배 (정확도 동일 가정)
- **시장 환경 적합도**: 폭락장에서 강력포착만 풀이면 적합도 0. 인버스 풀 있으면 적합도 1. **곱셈 효과 0 vs 1**.

복합 효과 (P5 vs P1): **100 × 2 × 5 × 1 = 1,000배** 일 기대 수익 (이론치).

현실에서는 사이즈 확대 시 슬리피지 ↑, 회전 ↑ 시 수수료 ↑ 등 마찰. **실현 가능 곱셈**은 1년 누적 **약 300~500배** 추정.

## 4. 정확도 측정 + 학습 루프 (자기반성 시스템)

수익 극대화의 두 번째 축은 정확도. 측정 안 하면 개선 불가.

### 4-1. 정확도 라벨링 자동화

매 거래 후 D+1, D+3, D+5 자동 라벨링:
```
data/reflection/{date}_{ticker}.json
{
  "entry": {"time": "14:00", "price": 50000, "size": 10},
  "exit": {"time": "14:45", "price": 50925, "rule": "TRAILING"},
  "label": {
    "D+1_close": 51500, "D+1_pct": +3.0, "D+1_hit": true,
    "D+3_close": 52000, "D+3_pct": +4.0, "D+3_hit": true,
    "D+5_close": 49500, "D+5_pct": -1.0, "D+5_hit": false
  },
  "signals": {"integrated_score": 92, "eye_pass": true, "regime": "MILD_BULL"},
  "context": {"market_pct": +0.8, "sector_fire_rank": 3}
}
```

### 4-2. 주간 회귀 분석 (금요일 16:30 자동)

매주 금요일 BAT-D 후 자동 분석:
- 적중률 (D+1, D+3, D+5)
- PF (Profit Factor)
- 시장 환경별 적중률 분포 (BULL vs NEUTRAL vs BEAR)
- 시간대별 적중률 (14:00 vs 14:30 vs 14:55 진입)
- 점수 구간별 적중률 (90~94 vs 95~100)
- 안전선 통과율 (어떤 안전선이 실제 손실 줄이는가)

결과 `data/reflection/weekly_{week}.md` 자동 작성, 사장님 카톡 요약 발송 (토요일 09:00).

### 4-3. 임계값 자동 조정 (월 1회)

매월 1일 회귀 결과 기반 임계값 미세조정:
- `integrated_score` 임계 (90 → 92 또는 88, 적중률 기반)
- 체결강도 임계 (현재 150, 검증 후 130~170 범위)
- VWAP 게이트 -2.5% (적합 시 -2.0% 또는 -3.0%)
- 거래량 폭증 임계 (현재 5x, 검증 후 3x~7x)

자동 PR 생성 → 사장님 카톡 → 승인 시 머지 → 다음 달 적용.

### 4-4. 단타봇 advisory 연계 (정보 공유)
