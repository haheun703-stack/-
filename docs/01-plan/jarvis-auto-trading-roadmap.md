## 5. 분할매수/매도 정확한 산식

### 진입 (5단 분할)

```python
target_qty = 100  # 목표 수량
entry_price = current_price

orders = [
    {"price": entry_price * 1.000, "qty": int(target_qty * 0.30)},  # 시장가 30%
    {"price": entry_price * 0.990, "qty": int(target_qty * 0.25)},  # -1% 25%
    {"price": entry_price * 0.980, "qty": int(target_qty * 0.20)},  # -2% 20%
    {"price": entry_price * 0.970, "qty": int(target_qty * 0.15)},  # -3% 15%
    {"price": entry_price * 0.960, "qty": int(target_qty * 0.10)},  # -4% 10%
]
```

### 청산 (4단 분할 + 손절)

```python
# 익절 (목표 도달 시)
exit_orders = [
    {"price": entry_avg * 1.05, "qty_pct": 0.30},  # +5% 30%
    {"price": entry_avg * 1.10, "qty_pct": 0.30},  # +10% 30%
    {"price": entry_avg * 1.15, "qty_pct": 0.25},  # +15% 25%
    {"price": entry_avg * 1.20, "qty_pct": 0.15},  # +20% 15%
]

# 손절 (즉시 전량)
stop_loss = entry_avg * 0.97  # -3%

# 트레일링 (수익 진행 시 손절선 이동)
trailing_rules = [
    (1.03, 1.00),  # +3% 도달 → 손절 진입가
    (1.05, 1.02),  # +5% 도달 → 손절 +2%
    (1.10, 1.05),  # +10% 도달 → 손절 +5%
    (1.15, 1.08),  # +15% 도달 → 손절 +8%
]
```

## 6. 자동 학습 메커니즘 (Phase 6)

```python
# 매 거래 종료 시 기록
record = {
    "date": "2026-05-19",
    "ticker": "005930",
    "signal": "PB15_BB",
    "entry_price": 270500,
    "exit_price": 285000,
    "ret_pct": 5.36,
    "holding_days": 3,
    "hit": True,  # 익절
    "stop_loss_hit": False,
}

# 시그널별 7일/30일 적중률 자동 계산
hit_rate_PB15_BB_7d = 0.65  # 적중률 65%
hit_rate_PB15_BB_30d = 0.58

# 가중치 자동 조정
if hit_rate_PB15_BB_7d < 0.50:
    weight_PB15_BB *= 0.9  # 10% 감소
elif hit_rate_PB15_BB_7d > 0.70:
    weight_PB15_BB *= 1.1  # 10% 증가
```
