## 1. 사고 요약

2026-06-03(수)은 **제9회 전국동시지방선거 임시공휴일**로 한국 증시 휴장.

pykrx 일괄조회가 6/3에 대해 **전종목 OHLCV=0 행**을 반환 → 그대로 parquet에 삽입.

- **수급 db**(nationality/investor): 6/3 행 0건 → 깨끗(휴장 필터 정상 작동)
- **kospi_index.csv / stock_data_daily CSV**: FinanceDataReader 기반이라 휴장 자동제외 → 깨끗

## 2. 핵심 구분 (중요)

**raw OHLCV·CSV는 깨끗한데 processed 지표 "저장본"만 오염**된 구조.
processed parquet은 지표 저장형(130개 컬럼: sma_*, rsi_14, macd, slope_ma60, ou_z 등)이라,
6/4 BAT-D가 6/3 0행이 시계열에 있는 상태로 지표를 계산·저장 → 6/4 행 지표 오염.
→ 6/3 행 제거만으로 부족, **지표 재계산 필수**.
