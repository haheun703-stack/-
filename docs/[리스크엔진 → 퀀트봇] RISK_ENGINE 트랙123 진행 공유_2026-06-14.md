## 2. 트랙별 상세

### ① 크라우딩 모니터 (`crowding.py`, §4.4)
- **게이트(G1~G8)가 아니라 L3 노출 조절 모니터.** pre_trade_gate의 PASS/RESIZE/REJECT가 아니라 `gross_exposure_mult`(1.0 또는 0.80)를 낸다 — 호출처가 사다리(§4.2)·변동성타겟팅(§4.3) 계수와 곱하는 구조.
- C1 보유 평균 쌍상관(60일) > `crowding_corr`(0.70) / C2 VKOSPI 20초과 **AND** 5일 30%급등(동시) / C3 외인선물 2년 분위수 상·하위 5%. 경고 **2개 이상** → −20%p(사다리와 별도). 3개여도 −20%p 고정.
- ★**C1 vs G5 혼동 주의**: C1 = 평시 동질성(균등가중 60일 피어슨), G5 = 위기 보수화 `ρ_stress`(슈링크). 목적이 달라 C1은 슈링크 미적용.
- C2/C3 시계열은 호출처 주입(미배선) → 미주입 = 미평가(과경고 차단). 견고성: C3 상수 `nunique` 가드(거짓 경고 차단, 적대검증으로 추가).

### ② Phase 3 나머지 (`vol_targeting.py` + `stress_test.py`)
- **변동성 타겟팅**(§4.3): EWMA halflife 20일 실현변동성 ×√252 → `scale = min(1.0, target 15%/realized)`. ★`scale ≤ 1.0` — 변동성 높으면 축소, 낮아도 레버리지 확대 안 함(§0 생존 목표).
- **스트레스 시나리오**(§4.1): **순수 집계 엔진**(사장님 승인). 역사 H1~H5(2008/2011/2020/2022/2024) + 가상 S1~S5. S3(KOSPI −8% 갭, 시장 베타 1 폴백)·S4(최대비중 하한가)는 비중만으로 **항상 평가**, S1(FX)·S2(반도체)·S5·H는 팩터 베타/역사 충격 주입 시 평가(미주입=미평가). `report.worst`로 최악 시나리오 조회.

### ③ G5 정밀화 — ρ_crisis (`correlation.py` + `gate_wiring`, §3.4)
- 슈링크 공식(`ρ_stress`)의 **대안** — VKOSPI 상위 10% 위기일만 모아 종목 간 동조화를 *실측*(가정 아님). `crisis_correlation_with` 추가, **기존 `stress_shrink`/`stress_correlation_with`는 무변경**.
- gate_wiring: `vkospi_series`(graceful, equity_peak_store 패턴) 주입 시 `ρ_effective = max(ρ_stress, ρ_crisis)` = "둘 중 나쁜 값"(§3.4).
- ★**정밀화는 G5를 절대 약화시키지 않음**(REJECT→PASS 불가, 강화만 가능). VKOSPI 미주입(현 production 미배선) → `crisis_map` 비어 슈링크만 = **G5 동작 불변, calibrated 보존**. 테스트가 `r_no` cluster==`[]` / `r_vk` cluster==`[H1]`로 격리 입증.
