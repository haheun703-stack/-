## 2️⃣ 차트 82장 학습 (정석 매매 45 + 3파/4파 37)

### 📘 정석 매매 영상 (45장)

**구성**: 인터뷰(인트로/마무리) + 코스피 주봉 박스권 + 한미반도체 일봉 단타

**핵심 매매법 2가지**:

#### A. 박스권 매매 (장기, 코스피 주봉)
```
박스 하단 분할 매수 → 박스 상단 분할 매도 → 한국 시장 다시 내려옴 → 또 모음
"지난 10년 거의 그랬으니까" — 통계적 근거 (backtest 10년 권장)
```

#### B. 단타 매매 (단기, 개별주 일봉)
```
매수: 신고가 돌파 → 눌림 → 20일선 근처 + 손익비 좋은 자리 분할
1차 매도: 이전 신고가 근처
2차 매도: 다음 신고가
3차 매도: 5일선 이탈 (트레일링)
최종 매도: 10일선 이탈 (전량)
재진입: 20일선 근처 → 사이클 반복 ("이거 반복이거든 저한테")
```

**영상 한 줄 결론**: "이런 종목 골라서 이때 사서 이때 팔아 — 딱 한 가지"

### 📗 3파/4파 상승 영상 (37장, Trading Factory)

**핵심**: 3파 놓쳤다면 4파 조정에서 5파 진입 잡기

**4파 진입 5/5 조건** (모두 충족 시 = "완벽한 눌림목 타점"):
1. **38.2% 피보나치 되돌림** 지지 (3파 시작점~끝점 전체 길이의 38.2%)
2. **파동 교대법칙**: 2파 V자 → 4파 횡보 / 2파 횡보 → 4파 V자
3. **4파 횡보형 + 매물 소화** (5파 임박 강력 신호)
4. ★ **4파 저점 > 1파 고점** (파동 중첩 금지 — 위반 시 패턴 무효)
5. 5/5 모두 충족 → 5파 진입

| 원칙 | 박스권 | 단타 | 3파/4파 |
|---|---|---|---|
| 분할 매수/매도 | ✅ | ✅ | ✅ |
| 추격 금지 | ✅ | ✅ | ✅ |
| 손익비 좋은 자리만 | ✅ | ✅ | ✅ |
| 반복 가능한 사이클 | ✅ | ✅ | ✅ |
| 한 방 노리지 않음 | ✅ | ✅ | ✅ |
