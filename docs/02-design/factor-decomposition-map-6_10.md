## 0. 팩터란

**팩터 = 여러 종목의 수익률에 공통으로 작용하는 수익의 원천(driver).**
핵심 질문: *"이 수익이 내 실력(알파)이냐, 이미 알려진 특성에 베팅한 결과(팩터 노출=베타)이냐?"*

- CAPM(시장 1팩터) → 파마-프렌치 3팩터(시장·사이즈·밸류) → +모멘텀·퀄리티·저변동성
- 팩터로 분해해야 **베타(이미 알려진 것)와 알파(진짜 내 실력)가 구분된다.**

## 2. 베타층 vs 알파층 (= "나만의 ETF" 2층 구조)

- **베타층(자산배분 — "어떤 팩터에 얼마")**: 거시 국면(C60 국면 + `macro_regime`) → 한국주식(FLOWX) / 레버·인버스(KODEX 헤지) / 현금(암묵 룰).
  - **미구축 갭**: 달러 · 금 · 해외 슬리브 (= FX·유동성 지시서 P1, 포트폴리오 OS 1단계)
- **알파층(종목선별 — "그 팩터 안에서 어떤 종목")**: 가설C(역추세+수급), 섹터 모멘텀, 밸류.
- **팩터 = 두 층을 가르는 자(尺).** 베타 구조(자산 간 상관)는 **6/1 "초강세장 검증불가"의 예외** — 장기 통계로 검증 가능.

## 3. 우리가 가진 것 / 갭

- **가진 것**: 모멘텀(추세 C60 ✅ + 섹터 신규 ✅ + 종목 RS), 밸류(부분), 수급(한국 특화), 거시, 저변동성
- **갭**:
  1. **퀄리티** — DART 영업이익만, ROE/부채/이익안정성 미합성
  2. **사이즈** — 시총 데이터는 있으나 팩터화 안 됨
  3. **자산배분(베타층)** — 달러·금·해외 미구축
  4. **★전략 간 / 자산 간 상관 미측정** (슈카영상이 짚은 핵심 갭 — "종류≠분산, 위기 때 상관 1 수렴")

## 4. 핵심 통찰

1. **우리는 이미 ~6개 팩터에 베팅 중이다** — 다만 흩어진 신호로 쓸 뿐, "어떤 팩터에 얼마나 노출됐나"로 **명시 분해·비중 관리는 안 하고 있다.**
2. **검증을 통과해 hard gate가 된 팩터는 C60(추세) 단독뿐.** 나머지(수급·거시·밸류·섹터)는 전부 관측 라벨 → 사후비교. (철학: 검증된 것만 게이트)
3. **수급·거시는 의도적으로 hard gate에서 배제** — 적대검증에서 noise/상시점등으로 판명(6/3). "신호가 많다"≠"알파가 많다".
