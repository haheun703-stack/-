# 거시 조기신호 적대검증 — negative 결과 = 방어 자산 (2026-06-03)

> **결론 한 줄**: 거시는 **지도**(시나리오 맥락), C60은 **핸들**(실행 타이밍). hard gate는 **self C60 단독** 유지.
> **상태**: shadow only / 실주문 0 / HOLD / 봇 OFF

## 배경

영상 논리(미국=정부 빚으로 AI·반도체 주가 몰빵, 회사채 스프레드 역사적 최저) →
"거시로 미리 약세전환을 피하자"는 유혹이 자연히 생긴다. 이 문서는 그 유혹을
**숫자로 검증해 막아둔** 기록이다. 어제 외국인 수급=NOISE 검증과 같은 종류의 방어 자산.

## 방법 (`scripts/research/macro_leadlag_adversarial.py`)

어제 `regime_obs_adversarial`와 동일: `base_rate / precision / honest_lead`.
- 대상: SOXL·TQQQ **self C60**(레버 자체 60선) 약세전환
- 신호: VIX>25 / 장단기역전(10Y−2Y<0) / 달러급등 / 금리급등(10Y) / HY스프레드
- **기간 2021-06~2023-06** — 약세장 '안'만 보면 VIX·역전이 내내 켜져(상시점등) 선행이 안 보이므로, 강세 끝물부터 봐서 "약세 진입 *전에* 켜졌나"를 측정.

## 박제 결론

- **hard gate 승격 거시 신호 = 0개.**
- **장단기 역전 = 상시점등 노이즈** (base 0.46). 통념상 침체 선행 지표지만 레버 타이밍엔 무력.
- **VIX = 민감하지만 거짓경보가 많아 hard gate 부적합** (recall 0.86 / precision 0.29 = 동반 신호이지 조기경보 아님).
- **달러·10Y 급등도 약함** (로그 관찰만).
- **HY 스프레드 = 2022 데이터 미확보**(이 환경 fdr/FRED는 2023-06~만 제공)라 **보류**. 영상 핵심 지표인데 미검증 — 향후 장기 데이터 확보 시 재검증.
- **거시 = 시나리오 맥락 / C60 = 실행 타이밍.** 역할이 다르다.
- **primary hard gate = self C60 단독 유지.**

## 한계 (정직)

- self C60 전환에 **휩쏘(톱니)가 섞여** 거시 신호가 불리 — "큰 약세장 전환만" 타겟하면 다를 수 있음(표본 부족).
- HY 스프레드 2022 미검증 (영상 핵심).
- 임계값(VIX>25, 달러 +3%/60d, 10Y +0.5%p/60d) 자의적.
- 표본 = 2021-06~2023-06, 본질적으로 약세장 1회.

## 의미

이 negative 결과 자체가 자산이다. 영상·내러티브에 흔들려 거시 신호를 hard gate에
넣었으면 **거짓경보로 손실**이 났을 것이다. 검증이 그걸 막았다 — 외국인 NOISE를
거른 것과 동일한 가치. 미래 시나리오 지도(A 몰빵지속 / B FOMC본게임 / C 패권균열)는
여전히 유효하되, **확인은 거시 맥락으로, 타이밍 실행은 C60으로** 역할을 나눈다.
