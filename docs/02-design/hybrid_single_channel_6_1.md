# 하이브리드 단일통로 설계 — "5~8봇 → 1통로" (6/1 사장님 통찰)

> 사장님 6/1: "1가지 엔진으로 대응 안 된다 → 하이브리드. 그전엔 5~8봇이 통제 없이 따로 놀았다.
> 기관·외인·금투·연기금이 *하는 짓*을 분석해서 진행하면 안 되나? 이거 계속 뺑뺑이 도는 느낌."
> ★ 이 문서 = 그 답. 흩어진 조각을 **통제 있는 단일 통로**로 통합하는 지도.

---

## 0. 진단 — 왜 뺑뺑이였나 (정직)
- 우리는 계속 **"단 하나의 예측기(IC>2)"**를 찾았다. 섹터로테이션 t=1.13, 수급 단순부호 IC=0, 점화 13번.
- 데이터는 매번 같은 답: **단독 신호는 약하다.** → 13번 돌았다.
- 진짜 문제 = 엔진 부족이 **아니라** 5~8봇이 각자 따로 예측기가 되려다 통합·통제가 없던 것.

## 1. 재구성 (핵심 사고전환)
> **예측기를 더 쌓지 않는다. 기관·외인·금투·연기금이 "하는 짓"을 *국면(context)*으로 읽고,
> 그 위에서 검증된 *실행 엣지(끼·손절+재진입)*로 돈을 번다 — 단 하나의 통로로.**

- smart-money 분석 = **미래 맞히는 예측기 ❌ / 지금 누가 사고 어디로 도는지 읽는 나침반 ✅**
- 돈은 나침반이 아니라 **실행**이 번다(손절+재진입 비용 5.0배 헤드룸·끼 단조 — 6/1 실측 검증).

## 2. 4층 하이브리드 (조각은 이미 다 있음 — 바닥부터 X)

```
        ┌───────────────── 단일 통로 (ONE channel) ─────────────────┐
        │  L1 국면나침반 → L2 명분 → L3 선정(끼) → L4 실행(돈)        │
        │           전부 명분게이트 + SAJANG 단일진실로 통제          │
        └────────────────────────────────────────────────────────────┘
```

| 층 | "기관·외인이 하는 짓" / 실행 | 현재 자산 | 역할 |
|---|---|---|---|
| **L1 국면(나침반)** | 11주체 수급(**금융투자=세력**·연기금·외국인·투신)·프로그램매매·섹터로테이션 | market_investor_collector(11주체, 6/1 배선)·flow_market·섹터로테이션(t=1.13) | regime(위험선호/회피)·섹터 핫맵·세력 잔류/이탈 → **context·필터**(사이징 X) |
| **L2 명분** | event_signals(재료) + 기존 44신호 | 명분게이트 `_aggregate_thesis_buckets`·매핑 v1·event_signals C-1 | "왜 오르나" 7요소 게이트 통과 |
| **L3 선정(끼)** | 끼 6팩터(검증된 수익원, 단조↑) | `score_kki`·4단 shadow | 高끼 × 명분통과 × L1 context |
| **L4 실행(돈)** | 맥점(눌림/돌파)·손절+재진입·줄먹 | 재진입 설계·cost harness(5x)·트레일링 | 줄먹은 여기서. **alpha의 원천** |

## 3. L1 국면나침반 — 사장님이 원한 "기관 분석" (핵심 신설)
"기관계 합계 샀다 → 산다"는 **이미 IC=0**(세력=금융투자를 합계가 가림). 그래서 L1은:
- **금융투자(세력) 라인 분리**: 11주체에서 금융투자 순매수 → 세력 매집/이탈 신호 (단순 기관계 X)
- **연기금·외국인**: 중·대형 가격형성 주체 → 대형주 regime
- **프로그램매매**: 차익/비차익 → 인덱스 압력·만기 효과
- **섹터로테이션**: 어느 섹터로 자금이 도나 (t=1.13 — context로만, 사이징 금지)
- **산출** = 종목별 "세력 잔류도" + 시장 "위험선호 regime" + "핫 섹터" → **명분게이트 수급버킷 보강 + 종목 필터**
- ★ **예측 아님**: L1은 종목을 고르지 않는다. L3 선정에 "지금 이 종목/섹터에 세력·외인이 있나"를 더해줄 뿐.

## 4. 단일 통로 = 통제 (5~8봇 문제의 답)
- **모든 신호는 명분게이트로 합류** → 끼 → 맥점 → 손절+재진입. 별도 봇이 독립 주문 ❌.
- **SAJANG 단일진실**: TP/SL/끼/명분/L1 임계 전부 `data/sajang_rules.py` 경유(리터럴 금지).
- **No Intent No Order**: 모든 매매 order_intent 1:1.
- **게이트 8/8 + 매도 무손상**: 모든 변경 후 회귀 0.
- → 5~8봇이 1 파이프라인. "통제 없이 따로 노는" 구조 종료.

## 5. ★ 뺑뺑이를 끊는 방법론 전환 (가장 중요)
| 버린다 | 채택한다 |
|---|---|
| per-signal IC>2 사냥 (13번 실패) | **통합 하이브리드의 손익비** forward 검증 |
| "이 신호가 미래를 맞히나?" | **"L1 국면필터를 켜면 L4 실행 손익비가 좋아지나?"** |
| 절대수익(생존편향 과대) | 상대비교(필터 on/off 손익비 차)·OOS·forward |

### 5b. ★ 예측 말고 실행으로 국면 적응 (6/1 퀀트 선정규칙 판정에서 도출)
퀀트 섹터선정규칙(섹터강도 top3 → 거래대금 top2 → 격주리밸) = **L1+L3 완성본**. 단:
- 후반 +94%p / **전반 -45%p / 전체 미달** + regime detector(언제 통하나) **이미 실패**(dispersion·IC지속성).
- → 라이브에서 "지금이 +94% 국면인지 -45% 국면인지" **사전에 모름** = 단독 매매 불가.

**해법 = regime detector를 더 찾지 않는다(또 실패). 손절(L4)로 빠져나온다:**
- 퀀트 규칙이 -45%p 먹는 이유 = **손절 없이 격주 보유**. 섹터 빠져도 다음 리밸까지 들고 있음.
- L4(고점-3% 트레일+재진입) 얹으면: 국면 돌면 -3% 손절이 MDD 막고, 살아있으면 재진입이 +94% 탐.
- **"국면을 예측해 피한다 ❌ / 실행으로 적응한다 ✅"** — IC 말고 손익비 철학의 핵심.
- **실험(hybrid 첫 실증)**: 퀀트규칙(L1+L3) + 손절+재진입(L4) 통합 손익비 — 특히 전반(-45%p)에서 손절이 MDD를 얼마나 막아 "미달→생존가능"으로 바꾸나.

- 검증 단위 = **돈(손익비·MDD·승률)**, IC 아님. (단타봇 1년 교훈: IC 유의 ≠ 수익)
- 절차: shadow(picks 불변, L1 context 기록·관측) → 손익비 리프트 측정 → paper → 사장님 승인 → live.

## 6. 정직한 경고 (과신 차단 — 14번째 뺑뺑이 방지)
1. L1조차 **단독 예측기로 믿으면 안 됨**. 금융투자 라인도 payoff 검증 전엔 context로만.
2. 섹터로테이션 t=2.29는 **사후분할·다중검정 미확인** → forward 사전등록 전 사이징 0.
3. 하이브리드도 "조각 많아져서 좋아 보이는" 과적합 위험 → **추가 층은 손익비 리프트 입증분만** 채택.
4. 절대수익·일봉·생존편향·표본 thin 한계 그대로 → 배수·상대비교만 신뢰.

## 7. 착수 순서 (shadow-first, 게이트 안전)
1. **L1 나침반 shadow**: 11주체→금융투자 세력라인 + 섹터핫맵 산출 → recommendation.json에 context 기록(picks 불변). jsonl 며칠 관측.
2. **L2 event_signals C-3**: 매핑 v1 → 재료버킷 shadow 배선(impact_score, 정보봇 freshness 답 후).
3. **통합 손익비 측정**: L1 context on/off로 L4(손절+재진입) 손익비 리프트 — backtest+forward.
4. 리프트 입증 + 게이트 8/8 + 매도 무손상 → paper → 사장님 승인 → live.
- **봇 OFF 유지 / 매도 무접촉 / picks 불변 / SAJANG 단일진실** 전 과정 불변.

## 8. 한 줄
> "기관·외인·금투·연기금이 하는 짓을 *나침반*으로 읽고(L1), 명분(L2)·끼(L3)로 고른 종목을,
> 맥점에서 사서 손절+재진입(L4)으로 줄먹는다 — **단 하나의 통제된 통로로.**
> 검증은 IC가 아니라 손익비로, forward로." → 이게 하이브리드고, 뺑뺑이의 끝이다.
