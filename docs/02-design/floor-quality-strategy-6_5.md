# 바닥 품질 필터 (floor_quality_score) 설계 — 2026-06-05

> 배경: 6/4 17:19 일산 서보성대리 통화("바닥을 어떻게 구하는가") + 6/5 데이터 검증
> 원칙: **"바닥 맞히는 계산기" 아님 → "바닥일 가능성 조건을 골라내는 필터"**. 확정 표현 금지.

## 1. 통화 핵심 (서보성대리, 미국장 전제)

1. **단순 물타기 = 무지(無知)**. 바닥은 **밸류에이션(PER/PBR/배당)으로 구한다**.
2. **저점 PER 밴드 × 현재 EPS = 바닥** — MS: 코로나·금리인상 저점 22배, 현재 380달러=23배 → "아직 진짜 싼 건 아님".
3. **삼성 = 배당 × 34배 = 최저 바닥** (역사 공식, 실제 49,900원 적중).
4. **떨어진 *이유*를 본다** — 시장 전체 조정이면 확신, 개별/구조 악재면 회피.
5. 바닥 확신 시 비중 ↑ (10~20% → 40%), 손절 max −15%.
6. (통화 본인 인정) **"한국은 자금(수급)에 따라 움직임이 많고, 미국은 밸류를 정확히 쳐준다"** → 미국 공식은 한국에 절반만 맞음.

## 2. 한국 적용 — 데이터 검증 결과 (6/5, fund 데이터 101종목)

- **삼성 PER 53(역사분위 100%) / SK PER 37(99%)** → AI 성장 대장주는 **밸류 바닥이 안 옴**. 조정(얕은 눌림) 매수만.
- **밸류 저점(PER&PBR 분위 30%↓) = 11종목**. 그중:
  - 🔴 **우하향(가치함정) 6개**: 한국전력(PER 2.9·−43% 우하향)·KT 등 = 규제·적자·산업쇠퇴 "싼 데는 이유 있음"
  - 🟢 **전환/상향(진짜 바닥 후보) 2개**: 현대제철·SK이노베이션 (밸류 저점 + 업황 반등)
  - 🟡 횡보 3개
- **밸류 단독이면 11개 다 매수 → 추세 필터로 6개 함정 제거.** 통화 4번("하락 이유")이 핵심임을 입증.
- 오늘 진입11 중 밸류 데이터 있는 2개(동진쎄미켐·이오테크닉스)는 둘 다 **밸류 극과열(99~100%)** → 6/5 반도체 조정 직격. "기술 눌림 ≠ 밸류 바닥".

## 3. floor_quality_score 스펙 (사장님 가이드 2026-06-05)

**공식**: `바닥 후보 = 밸류 저점 × 추세 둔화/전환 × 수급 회복 × 하락이유가 구조악재 아님`

| 요소 | 데이터 근사 | 적용 |
|------|------------|------|
| **밸류 저점** | PER/PBR 역사 분위(낮을수록 +) / 저점PER×EPS·배당×배수 바닥 대비 거리 | fund 있는 종목만 (없으면 중립) |
| **추세 둔화/전환** | 우하향(close<ma60<ma120)=관찰 / 횡보=후보 / 전환(close>ma60)=유망 | 전 종목 |
| **수급 회복** | 4주체(외국인·기관) 최근 순매수 전환 | 전 종목 |
| **★하락 이유** | KOSPI 대비 상대성과(시장조정 vs 개별) + 섹터 동조(업종조정) + 장기 우하향&밸류싼데안오름(구조악재 의심) | 전 종목 (가장 중요) |

**철칙 (사장님)**:
- 이름은 **`floor_quality_score`** 또는 `bottom_candidate_score`. "바닥 판정" 같은 **확정 표현 금지**.
- **hard gate 아님** — 차트 4조건(주봉LA·눌림지지·과열아님·RR)은 진입 게이트로 유지, floor_quality는 **보조 feature로 누적**.
- 처음부터 hard gate면 좋은 반등도 놓침.
- **6/12 사후비교**: 기술 바닥 후보 vs 밸류 바닥 후보 vs 밸류+추세 결합 — 뭐가 더 맞았나 숫자로.

**라벨(확정 X, 등급만)**: 우하향=`관찰(위험)` / 횡보=`바닥다지기 후보` / 전환=`진짜 바닥 후보`.

## 4. 한계 (정직)

- fund 데이터 **101종목(전체 9%)뿐** → 밸류 요소는 대형~중형 한정. 중소형은 추세·수급·이유만.
- "구조 악재" 완벽 판정 불가 → KOSPI/섹터 대비 + 추세로 **근사**. 6/12 검증으로 보정.
- 2026 AI 강세장은 밸류 극단(성장 프리미엄) → 통화도 "성장률 받히면 밸류 위로". 단순 PER 저점 적용 위험.

## 5. 이점

무작정 싼 종목 사는 실수를 줄이고, **반등 조건이 쌓이는 종목만 대기열에** 올림. "여기가 바닥 같다"를 감이 아니라 **조건**으로 말할 수 있게 됨.
