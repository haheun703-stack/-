### 0-B. STEP 5 산식 (실측 기반)

**TTM 영업이익** = 최신 Q누적 + 직전년도 사업보고서 - 직전년도 동일 Q누적
(예: 2025Q3 TTM = 2025Q3 + 2024Q4 - 2024Q3)

**CAGR_avg** = (CAGR_1y + CAGR_2y) / 2
- CAGR_1y = 최신 / 1년 전 동일 분기 - 1
- CAGR_2y = (최신 / 2년 전 동일 분기)^0.5 - 1
- 5년 지속 가정 클립: -30% ~ +50%

**미래 시총 (2 모델 병행 검증)**:
- **모델 A (PER 일정)**: 미래 시총 = 현재 시총 × (1 + CAGR_avg)^5
- **모델 B (5y_OP × 현재 PER)**: 미래 시총 = TTM_OP × (1 + CAGR_avg)^5 × 현재 PER

**업사이드 = (모델 A + 모델 B) / 2 / 현재 시총**

→ 두 모델 평균으로 단일 모델 편향 완화.

## 5. 본 분석의 한계 + 다음 보강

### 5-A. 데이터 한계
1. **PER 변동성**: KIS 트레일링 PER. 일회성 부진/호황으로 부풀려져 있음 (삼성전자 44.6, 효성중공업 71.9 등)
2. **CAGR 한계**: 1y + 2y 평균이지만 일회성 효과 완전 제거 X (LS의 CAGR 8.6%는 +50% 이전 base effect 영향)
3. **모델 A vs B 갭**: LS/SK하이닉스에서 큰 갭 → 추가 검증 필요
4. **PER 클립 부재**: 영업이익 적자 시 PER 음수/이상치 → 분석 불가 종목 2건

### 5-B. v1.1 갱신 항목 (정보봇 협업 또는 봇 자체 추가 작업)
- [ ] **산업 평균 PER**: 종목별 PER 대신 섹터 평균 PER 적용 → PER 변동성 완화
- [ ] **5년 추세 회귀**: 1y/2y 평균 대신 8분기 선형 회귀로 CAGR 추정
- [ ] **순이익 사용**: 영업이익 × PER → 순이익 × PER로 보정 (정확도 ↑)
- [ ] **TAM/SAM 적용**: 정보봇 STEP 2 시장규모 데이터로 미래 시총 상한 검증
