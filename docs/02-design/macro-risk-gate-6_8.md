## 4. 게이트 단계 (복합 조건, 절대 단독 금지)

| MacroRiskScore | 조치 |
|---|---|
| ≤ 1 | 기존 LIVE 조건 그대로 적용 |
| ≥ 2 (위험신호 2개+) | 신규 매수 축소 / **D0 종가 진입 금지, D1 검증 필요** |
| ≥ 3 (위험신호 3개+) | **freeze 유지 / D0 금지 / D1만 관찰 / LIVE 전환 보류** |

예시:
- US10Y 상승 + 환율 안정 + 외국인 매수 유지 → 그냥 금리 부담, 매도 근거 부족 (score 낮음)
- US10Y 상승 + 환율 급등 + 외국인 전기전자 매도 → 위험, 신규 진입 축소 (score≥2)
- + 지수 갭하락까지 → **Macro Risk Gate ON, D0 금지/LIVE 보류** (score≥3)

## 6.5 FX beta — 종목별 환율 민감도 모듈 (Macro Gate의 종목 차등)

Macro Risk Gate가 "시장 전체"라면, FX beta는 "종목별 차등"을 준다.

**계산**: `fx_beta` = 60거래일 회귀(종목 일수익률 ~ USDKRW 일변화율의 기울기). beta<0 = 환율↑에 하락(환율 인질), 절대값 클수록 민감.

**환율인질스코어** = fx_beta(민감도) × usdkrw_trend(환율 추세) × foreign_net_5d(외국인 수급)

**신호 2개**:
- **회복 후보**: 환율 하락 전환 + fx_beta 큰 음수 + 외국인 매수 전환 → "환율 풀려서 반등 시작"
- **경고**: 환율 상승 지속 + fx_beta 큰 음수 + 외국인 매도 → "비중 조절"

**6/8 실측 (60거래일)**:
| 종목 | fx_beta | 상관 | R²(환율 설명력) |
|---|---|---|---|
| SK하이닉스 | −2.55 | −0.35 | 12% |
| 삼성전자 | −2.39 | −0.37 | 14% |
| LG화학 | −2.11 | −0.35 | 12% |
| 현대차 | −2.09 | −0.32 | 10% |
| LG엔솔 | −1.56 | −0.30 | 9% |
| NAVER | −1.29 | −0.27 | 7% |

→ 한국 대형주 전부 환율 인질(음수). 반도체>현대차>NAVER 순. **NAVER 둔감(−1.29)**이 6/8 반도체 패닉 속 디커플링 급등을 설명.

**현대차 사례 (모듈의 동기)**: 종목 악재 0, 호재만. 외국인 6월 순매도(★**6/2부터** — 5/29~6/1은 환율↑에도 현대차 +10.8% 급등·외국인 0 = 환율 단독동인 아님)가 환율(1495→1533, +2.6%) 디레버리징과 정합. **환율은 현대차의 핵심 동인 중 하나(단독 아님).** 환율 꺾이면 beta −2.09 방향으로 반등 여지, 더 오르면 외국인 매도 압력 지속 — 단 R² 10%이라 환율만으론 단정 불가, 수급·글로벌 리스크오프와 함께 관찰.

**적대적 경계**:
- **R² 7~14%** = 환율은 종목 변동의 7~14%만 설명. **단독 사용 금지.**
- **전부 음수** = 시장 공통 요인이라 종목 선별 차별성 약함(beta 크기 순위로만).
- 환율 방향도 **추세 추종이지 예측 아님**.

**★정보봇 교차검증 (2026-06-08)** — 방향은 검증, 단정은 완화:
