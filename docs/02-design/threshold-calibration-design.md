## 3. 핵심 한계 + 우회 전략 (정직 고백)

| 한계 | 우회 |

## 4. 산식 정의

### (3) D+1 적중률 ≥ N% 승급 기준
```python
ret_d1 = (close[t+1] - close[t]) / close[t]
hit_d1 = ret_d1 > 0  # 양봉
acc_d1 = mean(hit_d1[signal_triggered])
```
**sweep**: 40%, 45%, 50%, 55%, 60%, 65%, 70% (7개) × 시장 레짐 3종 (STRONG_BULL / MILD_BULL / NEUTRAL) = 21개 셀
**평가 지표**: 표본 수 + 누적 수익률 + PF + MDD

### (4) 진입 가드 매트릭스
```python
trigger = (
    vol_ratio >= V        # V ∈ {1.5, 2.0, 3.0}
    AND foreign_net >= F  # F ∈ {0, 10억, 50억}
    AND inst_net >= I     # I ∈ {0, 10억, 50억}
)
```
3×3×3 = **27개 조합** sweep, 각각 D+1/D+3/D+5 적중률 + PF 산출.
**선정 기준**: PF ≥ 1.5 AND 표본 수 ≥ 50 AND D+3 적중률 ≥ 55%

### (2) 거래량 임계값
```python
# 5분봉 기반
vol_5min_ratio = vol_5min[t] / mean(vol_5min[t-5d, t-1d, 동일 시간대])
```
**sweep**: 1.5x, 2x, 3x, 5x, 7x, 10x (6개) × 시간대 3종 (09:00-10:00 / 10:00-14:00 / 14:00-15:20) = 18개 셀

### (1) 체결강도 근사
```python
# 5분봉 매수압력 근사
buy_pressure = sum(vol[i] × sign(close[i] - open[i]) × (close[i] - open[i]) / open[i])
              for i in 5분봉_3개 연속  # 약 15분 윈도
# 표준화: percentile rank vs 직전 60거래일 동일 시간대
strength_pct = percentile_rank(buy_pressure)
```
KIS `tday_rltv` 150 ↔ percentile 약 95~99 추정. 분봉으로 검증 후 매핑.

## 5. 백테스트 로직

1. **데이터 적재** (`scripts/calibrate_thresholds.py`)
   - 정보봇 2,632종목 CSV → pandas (시총 상위 500종목으로 1차 제한)
   - 자체 5분봉 → 5/18~ 일별 폴더 순회
2. **시그널 평가**: 각 거래일 × 각 임계값 조합 → 진입 여부 boolean
3. **라벨링**: D+1/D+3/D+5 종가 → 수익률 계산 (수수료 0.015% × 2 + 거래세 0.18% 차감)
4. **집계**: 임계값별 표본 수 / 적중률 / 평균수익률 / PF / MDD
5. **분할**: 시장 레짐 (Regime_Tag) × 시총 구간 × 종목 유형 (ETF/대형주/중형주)
