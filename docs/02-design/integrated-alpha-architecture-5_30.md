## 3. 3파4파 작전법 (d:\...\3파,4파 상승\)

엘리어트 파동 **4파 눌림목 매매법**(동영상 자막+차트29장, 코드 미통합):
- 3파(최강 상승) 추격 대신 → **4파 조정에서 5파 진입**
- 기준: ①피보나치 38.2% 되돌림 지지 ②파동 교대(2파 V자면 4파 횡보) ③파동중첩 금지(4파 저점 > 1파 고점)
- 4파가 횡보(매물소화) = 5파 임박

**평가**: 새 작전이 아니라 **진입 타이밍 기법**. 우리 6원칙이 밝힌 "폭등 시작 RSI 72~83=추격은 늦음"의 해법. stock-analyzer의 "1차분출 후 눌림목 재돌파(SET-1)"와 동일 개념. → **종목 선정(외인+섹터)은 우리 엔진, 진입 타이밍은 4파 눌림**으로 결합.

오늘 발견(외인단독 PF1.88·수급D20·6원칙) + 기존 엔진(거시흐름·sector_fire·릴레이·4파)을 **하나로 연결**:

```
[거시] investor_flow 레짐 (외인 선별매수 국면)
  → [섹터] sector_fire 발화(액침냉각 A) + group_relay 릴레이(선도→후행)
  → [종목] 외인 단독 매집(PF1.88) + 수급 D+20
  → [타이밍] 4파 눌림(피보38.2%)/SET-1 — 추격 회피

  → approved_selector (source_weight 가중)
  → order_intents → paper 실행
  → 성과 → signal_ledger → source_weight 학습 (루프)
```

**= 거시→섹터→종목→타이밍 6원칙을 단일 파이프라인으로.** 이게 코덱스 청사진(FLOWX)의 실현.
