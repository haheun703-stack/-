# 상한가 수급선행 신호화 백테스트 — 판정: ❌ 기각

> ⚠️ **적대검수 각주 (2026-07-07 밤)**: D+5 중첩 수익률의 일별 t는 자기상관 미보정으로 ~√5배 과대 —
> 외인3~5 t=2.30→실질 ~1, 쌍끌이3 조정장 t=-2.89→실질 ~-1.3(유의 아님). 어느 방향도 유의 없음 =
> 기각 결론은 오히려 강화. 데이터 창: 로컬 processed 6/19 종료.

**실행일**: 2026-07-07 | **기간**: 2025-04-21 ~ 2026-06-18 (266,809 종목일·1,160종목·거래대금 5억+ 필터)
**질문**: 사후학습의 "상한가 53%가 D-1 수급선행"(재현율)을 뒤집어 — streak 보유 종목을 익일 시가에 사면 되는가?

## 1. 상한가 예측력: 없음 (재현율≠정밀도 확정)

| 신호 | n | P(익일 +25%) |
|---|---|---|
| 기저율(전체) | 266,809 | **0.35%** |
| 쌍끌이1 (사후학습 정의) | 24,219 | 0.37% |
| 쌍끌이3 | 2,091 | 0.33% |
| 과매집(f≥8) | 3,030 | 0.17% |

수급선행 종목의 익일 상한가 확률은 기저율과 동일. **"53%"는 상한가에서 역추적한 재현율일 뿐, 신호 가치 없음.**

## 2. 초과수익 헤드라인은 커버리지 편향이었다 (★방법론 교훈)

- 1차 결과: 외인3~5 D+5 초과 +0.54%p(t=5.1) — 그럴듯해 보였음.
- **교란 발견**: 수급 적재 종목일 비율이 2026-03부터 25%→52%→92%→100% 급증(6/27 KIS 11세분 백필).
  초기 구간의 대조군에 '수급 미적재 종목'(성과 상이한 소형주)이 섞여 초과수익이 부풀려짐.
- **수급 적재 종목만 대조군**으로 재계산:

| 신호 | 강세(25/04~26/02) | 조정(26/03~06) |
|---|---|---|
| 외인 3~5 밴드 | +0.24%p (t=+2.30) | +0.21%p (t=+1.44) |
| 쌍끌이3 | +0.76%p (t=+2.03) | **-1.21%p (t=-2.89)** |
| 쌍끌이1 | +0.22%p (t=+2.23) | -0.21%p (t=-1.39) |

## 3. 결론

1. **신호화 기각**: 최대 초과 +0.2%p/5d 수준 — 왕복비용 0.35%에 잠식. 쌍끌이는 조정장에서 유의한 역효과.
2. 사후학습 트랙은 본래 목적('수급 선행했는데 놓친 상한가' 원인 태깅)으로만 유지. 가점 배선 안 함.
3. ★교훈 2건: ①재현율(상한가→수급)과 정밀도(수급→상한가)를 혼동 말 것 ②**대조군은 데이터 커버리지 동질 집단으로** — 커버리지 변화 시점(백필·수집 전환)을 반드시 확인.
