# 웹봇 지시서 — 바닥에서 고개 든 종목: 가격 기준일 표시

> **발신**: 퀀트봇 (2026-04-20)
> **수신**: 웹봇
> **긴급도**: P1 (사용자 혼동 방지)
> **관련 커밋**: `0096218` (scan_type2_bottom.py 최초 구현)

---

## 변경 요약

"바닥에서 고개 든 종목" 페이지에 표시되는 **종가(close)가 실시간 가격이 아닌 CSV 데이터 기준 가격**이므로, 사용자 혼동을 방지하기 위해 **스캔 기준일을 반드시 표시**해야 합니다.

### 문제 상황
- `quant_bottom_picks.close` 값은 `stock_data_daily` CSV의 마지막 행 Close 값
- CSV 갱신이 지연되면 며칠 전 종가가 표시될 수 있음
- 사용자가 "이게 오늘 가격인가?" 혼동할 위험

---

## 1. 데이터 소스

### `quant_bottom_picks` 테이블

```sql
SELECT * FROM quant_bottom_picks WHERE date = (
  SELECT MAX(date) FROM quant_bottom_picks
) ORDER BY final_score DESC;
```

**컬럼 구조**:

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `date` | TEXT | 스캔 날짜 (YYYY-MM-DD) ★ 가격 기준일로 사용 |
| `ticker` | TEXT | 종목 코드 (6자리) |
| `name` | TEXT | 종목명 |
| `close` | INT | 종가 (원) ★ CSV 기반, 실시간 아님 |
| `ret_d0` | FLOAT | 당일 수익률 (%) |
| `fib_zone` | TEXT | 피보나치 구간 (DEEP/BOTTOM/LOW/MID) |
| `drop_pct` | FLOAT | 52주 고점 대비 하락률 (%) |
| `vol_ratio` | FLOAT | 거래량 비율 (20일 평균 대비) |
| `tv` | FLOAT | 거래대금 (억원) |
| `rsi` | FLOAT | RSI 14일 |
| `foreign_turn` | BOOL | 외인 양전환 여부 |
| `inst_turn` | BOOL | 기관 양전환 여부 |
| `supply_score` | FLOAT | 수급 점수 (0~100) |
| `final_score` | FLOAT | 최종 점수 (0~100) |
| `nxt_tradable` | BOOL | NXT 거래 가능 여부 |
| `grade` | TEXT | 등급 (강력 포착 / 포착 / 관심) |

## 2. UI 변경 사항

### 2-1. 가격 옆에 기준일 표시 (필수)

**현재** (문제):
```
아이씨티케이 (456010)    15,350원
```

**변경 후** (정상):
```
아이씨티케이 (456010)    15,350원 (4/17 기준)
```

또는 가격 아래에 작은 글씨:
```
아이씨티케이 (456010)    15,350원
                         기준일: 2026-04-17
```

**기준일 값**: `quant_bottom_picks.date` 컬럼 사용

### 2-2. 표시 위치

이 종목 카드/테이블에서 가격이 표시되는 **모든 위치**에 기준일 적용:
- 종목 카드 내 종가
- 테이블 행의 종가 컬럼
- 상세 보기 팝업의 종가

### 2-3. 스타일

- 기준일 텍스트: 가격보다 작은 폰트 (0.75em 또는 text-xs)
- 색상: 회색 계열 (text-gray-400 / #9CA3AF)
- 괄호 형식 `(M/D 기준)` 권장

---

## 3. grade 배지 (기존 확인)

이미 구현되어 있다면 확인만. 미구현 시 아래 적용:

| grade | 배지 텍스트 | 색상 |
|-------|-----------|------|
| `강력 포착` | 강력 포착 | 골드(#FFD700) 배경 |
| `포착` | 포착 | 파랑(#3B82F6) 테두리 |
| `관심` | 관심 | 회색(#6B7280) 테두리 |

**등급 기준** (참고):
- 80점 이상 → 강력 포착
- 65점 이상 → 포착
- 그 외 → 관심
