## 2. 피보나치 페이지 UI 변경

### 2-1. 상단 하이라이트 섹션 (신규)

`entry_picks` 배열이 1건 이상일 때, 피보나치 페이지 최상단에 표시:

```
┌─────────────────────────────────────────────────────┐
│  ★ 매수 적기 3종목 — RSI 과매도 + 수급 유입          │
│  백테스트 근거: D+5 수익률 +2.97%, 승률 61.9%       │
├─────────────────────────────────────────────────────┤
│  ★ NAVER (035420)      189,000원  ▼30.5%           │
│    RSI 38.2 | 외인 +88억 | 기관 +13억              │
│    목표가 245,000원 (▲29.6%)                        │
│                                                     │
│  ★ LG전자 (066570)     82,000원   ▼25.3%           │
│    RSI 35.1 | 외인 +45억 | 기관 +22억              │
│    목표가 108,000원 (▲31.7%)                        │
│                                                     │
│  ...                                                │
└─────────────────────────────────────────────────────┘
```

**스타일 가이드**:
- 배경: 기존 FLOWX 다크 테마에 맞춰 골드/앰버 계열 강조
- ★ 아이콘: 골드(#FFD700) 또는 앰버(#F59E0B) 색상
- 하락률(drop): 빨간색 텍스트
- 목표가 상승여력(upside): 초록색 텍스트

> **[중요] 가격 기준일 표시 필수**
>
> 종목 가격 옆에 **스캔 기준일**을 반드시 표시해야 합니다.
> 가격이 며칠 전 데이터일 수 있으므로, 사용자가 "이게 오늘 가격인가?" 혼동하지 않도록 합니다.
>
> 예시: `189,000원 (4/17 기준)` 또는 가격 아래에 작은 글씨로 `기준일: 2026-04-17`
>
> 기준일 데이터는 `quant_fib_scanner.date` 또는 `quant_jarvis.date` 값을 사용합니다.

### 2-2. 기존 종목 카드에 `entry_grade` 배지 추가

각 종목 행에 `entry_grade` 값에 따른 배지 표시:

| entry_grade | 배지 텍스트 | 색상 | 의미 |
|-------------|-----------|------|------|
| `"적기"` | ★ 적기 | 골드(#FFD700) 배경 | RSI<40 + 수급 유입 |
| `"관심"` | 관심 | 파랑(#3B82F6) 테두리 | RSI<40만 충족 |
| `"수급 유입"` | 수급↑ | 초록(#10B981) 테두리 | 수급만 충족 |
| `"대기"` | *(표시 안 함)* | — | 조건 미충족 |

배지 위치: 종목명 오른쪽 또는 가격 옆 (기존 grade 배지와 나란히)

### 2-3. RSI + 수급 컬럼 추가 (선택사항)

기존 테이블에 RSI / 외인 / 기관 컬럼을 추가 표시하면 더 좋음:

| 종목 | 가격 (기준일) | 하락률 | RSI | 외인(억) | 기관(억) | 적기 |
|------|-------------|--------|-----|---------|---------|------|
| ★ NAVER | 189,000 (4/17) | -30.5% | 38.2 | +88.3 | +12.5 | ★ 적기 |

| 카카오 | 38,500 (4/17) | -42.1% | 28.5 | -5.2 | +8.3 | 관심 |

## 3. 퀀트시스템 메인 UI 변경

### 3-1. 피보나치 매수 적기 섹션 (신규)

`quant_jarvis.data.fib_entry_picks` 배열이 1건 이상일 때 표시.

**위치**: 퀀트시스템 탭 내 적절한 Row (기존 Row4 부근 또는 별도 Row)

```
┌─────────────────────────────────────────────────┐
│  📐 피보나치 매수 적기                    3종목  │
│  RSI 과매도 + 수급 유입 동시 충족                │
├─────────────────────────────────────────────────┤
│  ★ NAVER       189,000원  RSI 38  외인 +88억   │
│  ★ LG전자       82,000원  RSI 35  외인 +45억   │
│  ★ SK하이닉스  160,000원  RSI 39  기관 +30억   │
│                                                 │
│  → 피보나치 페이지에서 상세 보기                │
└─────────────────────────────────────────────────┘
```

**표시 규칙**:
- `fib_entry_picks` 배열이 비어있으면 해당 섹션 자체를 숨김
- 최대 5종목까지 표시 (초과 시 "외 N종목" 표기)
- 하락률 큰 순서 (이미 정렬되어 있음)
- "피보나치 페이지에서 상세 보기" 링크로 해당 페이지 연결

## 4. entry_grade 판정 기준 (참고)

```
RSI < 40 (과매도) + 외인/기관 순매수 ≥ 5억원  →  "적기"
RSI < 40만 충족                                →  "관심"
수급만 충족 (외인/기관 ≥ 5억)                  →  "수급 유입"
둘 다 미충족                                   →  "대기"
```

- RSI: 14일 기준 상대강도지수, 40 미만 = 과매도 구간
- 수급: 외국인 또는 기관 순매수 중 하나가 5억원 이상
- 백테스트 근거: S7 전략 (Fib 0.618 + RSI<40 + 수급) = D+5 수익률 +2.97%, 승률 61.9%, PF 3.00
